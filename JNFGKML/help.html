Comment utiliser l'application

🎨 Thèmes

Cliquez sur les icônes en haut de la page pour changer le thème :

  • 🌞 Thème par défaut
  • 🌙 Thème sombre
  • 🎨 Thème pastel
  • ✨ Thème néon

💌 Messages d'amour

Dans la section "Nos Messages d'Amour" :

  • Écrivez votre message dans le champ de texte
  • Cliquez sur "Envoyer" ou appuyez sur Entrée pour l'envoyer
  • Utilisez le bouton 🗑️ pour supprimer un message

📸 Photos

Dans la section "Nos Photos" et "Roulette des Photos" :

  • Cliquez sur "Nouvelle Photo" pour voir une photo aléatoire
  • Utilisez "Ajouter des Photos" pour ajouter vos propres photos
  • Utilisez le bouton 🗑️ pour supprimer une photo

🤝 Connexion P2P

Pour partager l'application avec votre partenaire :

  • Cliquez sur "Démarrer une nouvelle connexion"
  • Un ID de connexion unique sera généré automatiquement
  • Partagez cet ID avec votre partenaire
  • Votre partenaire peut coller cet ID pour se connecter
  • Une fois connecté, vous pourrez partager :
    • 💌 Messages d'amour
    • 📸 Photos
    • 🎨 Thèmes
